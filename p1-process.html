📥 ขั้นตอนที่ 1: รับข้อมูลผู้สมัคร

วัตถุประสงค์: ดึงข้อมูลผู้สมัครทั้งหมดจาก Google Sheet (BCT PPP7)
แหล่งข้อมูล: Sheet "A1_ข้อมูลผู้สมัครทั้งหมด" — ตาราง Registerform
หมายเหตุ: ระบบจะตัดใบสมัครซ้ำออกอัตโนมัติ
💡 วิธีดึงข้อมูล:
1. เปิด Apps Script ของ Sheet: คลิกที่นี่
2. Extensions → Apps Script
3. Copy function exportApplicantsToAutoSync() ไปวาง
4. กด Run → ข้อมูลจะเข้าระบบอัตโนมัติ

กำลังโหลดข้อมูล

🔍 ขั้นตอนที่ 2: คัดเลือกเบื้องต้น

วัตถุประสงค์: HR คัดเลือกผู้สมัครที่มีคุณสมบัติตรงกับตำแหน่ง
เกณฑ์: ตำแหน่งตรง / อายุเหมาะสม / วุฒิการศึกษาผ่าน / ประสบการณ์เกี่ยวข้อง

📝 ขั้นตอนที่ 3: ส่งแบบทดสอบ (4 ชุด)

วัตถุประสงค์: ส่งลิงก์แบบทดสอบให้ผู้สมัครทำออนไลน์
แบบทดสอบ: 1. ทัศนคติ 2. ทักษะ 3. ทุนองค์กร (CC) 4. 3E3P
วิธีส่ง: สร้างลิงก์ → Copy ข้อความ → ส่ง LINE/Facebook

🤖 ขั้นตอนที่ 4: AI วิเคราะห์คุณสมบัติ

วัตถุประสงค์: AI วิเคราะห์ผลแบบทดสอบ + ข้อมูลผู้สมัคร
AI ทำ: วิเคราะห์จุดแข็ง-จุดอ่อน, คะแนน CC แต่ละด้าน, แนะนำ pass/fail

✅ ขั้นตอนที่ 5: คัดเลือกเข้าสัมภาษณ์ (P2)

วัตถุประสงค์: ผู้ปฏิบัติงานเลือกผู้สมัครที่ผ่านเกณฑ์เข้าสัมภาษณ์ P2
เกณฑ์: คะแนนแบบทดสอบผ่าน + AI แนะนำ "ผ่าน"

📩 ขั้นตอนที่ 6: ส่งข้อความนัดสัมภาษณ์

วัตถุประสงค์: ส่งข้อความนัดสัมภาษณ์ให้ผู้สมัคร
เนื้อหา: วันที่, เวลา, สถานที่, เอกสารที่ต้องเตรียม

🧠 ขั้นตอนที่ 7: AI สร้างแบบสัมภาษณ์อัตโนมัติ

วัตถุประสงค์: AI สร้างคำถามสัมภาษณ์ตามตำแหน่ง + ข้อมูลผู้สมัคร + ผลแบบทดสอบ

🎤 ขั้นตอนที่ 8: สัมภาษณ์ (5 ผู้สัมภาษณ์)

วัตถุประสงค์: 5 ผู้สัมภาษณ์ให้คะแนน
ผู้สัมภาษณ์: 1. HR 2. หัวหน้างาน 3. เพื่อนร่วมทีม 4. ลูกค้าภายใน 5. ผู้บริหาร

📊 ขั้นตอนที่ 9: AI สรุปผลและให้ความเห็น

วัตถุประสงค์: AI สรุปผลการสัมภาษณ์ + แนะนำรับ/ไม่รับ + เหตุผล
