📊 สถิติกระบวนการ

👥 รายชื่อผู้สมัคร

🔄 Sync ข้อมูลระหว่างผู้ปฏิบัติงาน

ระบบนี้เก็บข้อมูลในเบราว์เซอร์แต่ละเครื่อง ใช้ฟีเจอร์นี้เพื่อ sync ข้อมูลระหว่างกัน

📤 Export ข้อมูล (ส่งให้คนอื่น)

คลิกปุ่มด้านล่างเพื่อ export ข้อมูลทั้งหมดเป็นไฟล์ JSON แล้วส่งไฟล์ให้ผู้อื่น

📥 Import ข้อมูล (รับจากคนอื่น)

เลือกไฟล์ JSON ที่ได้รับจากผู้อื่นเพื่อ import เข้าระบบ


💡 วิธีใช้งาน

  1. Admin เพิ่มข้อมูลผู้สมัคร
  2. Admin คลิก "Export ข้อมูลทั้งหมด" → ได้ไฟล์ JSON
  3. ส่งไฟล์ JSON ให้ผู้ปฏิบัติงานคนอื่น (ทาง LINE, Email, etc.)
  4. ผู้ปฏิบัติงานคนอื่นคลิก "Import ข้อมูล" → เลือกไฟล์ JSON
  5. ข้อมูลจะ sync กันทั้งหมด
