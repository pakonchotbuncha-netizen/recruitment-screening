# กระบวนการคัดกรองและสัมภาษณ์ตำแหน่งงาน
## Recruitment Screening & Interview Process

ระบบคัดกรองและสัมภาษณ์ผู้สมัครงาน แยกจาก Web PP7

---

## 🎯 วัตถุประสงค์

สร้างระบบคัดกรองผู้สมัครงานแบบครบวงจร ตั้งแต่รับใบสมัครจนถึงสรุปผลการสัมภาษณ์
โดยใช้ AI ช่วยวิเคราะห์และสร้างแบบประเมินอัตโนมัติ

---

## 📋 กระบวนการ 9 ขั้นตอน

### Step 1: รับข้อมูลผู้สมัคร
- ดึงข้อมูลจาก Google Sheet (ใบสมัครทั้งหมด)
- แสดงรายชื่อผู้สมัครทั้งหมดในระบบ

### Step 2: คัดเลือกเบื้องต้น
- ตรวจสอบคุณสมบัติพื้นฐาน
- กรองผู้สมัครที่มีคุณสมบัติตรงตามตำแหน่ง

### Step 3: แบบทดสอบออนไลน์
- แบบทดสอบทัศนคติ (Attitude Test)
- แบบทดสอบทักษะ (Skill Assessment)
- แบบทดสอบทุนองค์กร (Corporate Capital - CC)
- แบบทดสอบ 3E3P

### Step 4: AI วิเคราะห์คุณสมบัติ
- AI ประเมินผลจากแบบทดสอบ
- วิเคราะห์ความเหมาะสมกับตำแหน่ง
- สร้างรายงานสรุปจุดแข็ง/จุดอ่อน

### Step 5: คัดเลือกเข้าสัมภาษณ์ P2
- ผู้ปฏิบัติงานทบทวนผลการวิเคราะห์ของ AI
- คัดเลือกผู้สมัครที่เหมาะสมเข้าสัมภาษณ์

### Step 6: ตอบรับการสัมภาษณ์
- ส่งคำเชิญสัมภาษณ์ให้ผู้สมัคร
- ผู้สมัครยืนยันการเข้าร่วม

### Step 7: AI สร้างแบบสัมภาษณ์
- สร้างคำถามสัมภาษณ์อัตโนมัติ
- รวมข้อมูลผู้สมัคร ผลแบบทดสอบ
- ปรับคำถามตามตำแหน่งและระดับ

### Step 8: กระบวนการสัมภาษณ์
- ผู้สัมภาษณ์ 5 คนให้คะแนน
- ประเมินตามเกณฑ์ CC และทักษะ
- บันทึกคะแนนและความคิดเห็น

### Step 9: AI สรุปผลการสัมภาษณ์
- สรุปคะแนนจากผู้สัมภาษณ์ 5 คน
- AI วิเคราะห์และให้ความเห็น
- สร้างรายงานสรุปสำหรับผู้ตัดสินใจ

## 🔧 เทคโนโลยี

- **AI:** OpenAI API / Claude API (สำหรับวิเคราะห์และสร้างคำถาม)
- **Deploy:** GitHub Pages / Google Web App
